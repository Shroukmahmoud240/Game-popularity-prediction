# 🎮 Game Popularity Prediction

This project is a Machine Learning-based system for predicting game popularity using both **Regression** and **Classification** models. It also includes a simple **GUI** for easy interaction and testing.

## 📌 Project Overview

The goal of this project is to analyze game-related features and predict:

- 🎯 Game popularity score (Regression)
- 🏷️ Popular / Not Popular classification (Classification)

The project demonstrates the full ML pipeline:
data preprocessing → feature engineering → model training → evaluation → GUI deployment.

---

## 🧠 Models Used

### Regression Model
- Predicts continuous popularity score
- Evaluated using:
  - MAE
  - MSE
  - R² Score

### Classification Model
- Predicts whether a game is popular or not
- Evaluated using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
